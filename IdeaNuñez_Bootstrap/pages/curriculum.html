Curriculum Vitae

• Pruebas de Software de tipo funcional, no funcional, caja blanca y asociado al cambio. • Desempeño en los cuatro niveles de testing: pruebas de componente, pruebas de integración, pruebas de sistema y pruebas de aceptación. • Gestión y documentación de defectos encontrados: defectos de contenido, defectos funcionales, defectos disruptivos, elaboración de reporte de bugs. • Pruebas estáticas y análisis de documentación tales como: plan maestro de pruebas, casos de uso, criterios de aceptación, manuales, análisis funcional, requerimientos funcionales, historias de usuario. • Planificación de pruebas: alcance, medios y riesgos de las pruebas, enfoque general de las pruebas, definición de calendario de actividades de análisis, así como el diseño, implementación, ejecución y evaluación de las mismas. • Web testing: pruebas de compatibilidad en diferentes browser, pruebas de cookies, pruebas de links. pruebas de UX/UI, pruebas de web services, responsividad. • Mobile Testing: Pruebas en Android, Android Visual Studio, Logcat. Pruebas en IOS: Xcode, Instruments, Bitrise. • API Testing: prueba de micro servicios, prueba de endpoints, pruebas en API SOAP y REST. • Pruebas de stress, verificación de estabilidad, picos de conectividad, load testing, endurance testing, scalability testing. • Conocimientos en Metodología Agile Scrum. • Manejo de lenguaje Java Script, HTML5, CSS3. Manejo de Software: Selenium IDE Jmeter Appium Atlassian Jira SOAP UI Postman
